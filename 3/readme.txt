1. В массиве хранятся значения высот профиля местности с постоянным шагом по горизонтали. Найдите области (точки высот), невидимые для наблюдателя, находящегося в первой точке.

2. Новобранцы выстроены в шеренгу. По команде НАЛЕ–ВО! одни из них поворачиваются налево, а другие – направо. Это есть начальное положение шеренги. Далее каждый новобранец действует следующим образом: увидев перед собой лицо соседа, он ровно через секунду поворачивается кругом. В конце концов повороты прекратятся.
